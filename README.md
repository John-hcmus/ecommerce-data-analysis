# ecommerce-data-analysis

This project analyzes sales data from an ecommerce business using SQL and Python.
The goal is to extract business insights related to revenue, customers, and products.

## Habit Interview app

A lightweight front-end app is included at `app/`.

### Run locally

```bash
cd app
python3 -m http.server 4173
```

Open `http://localhost:4173` and complete the guided interview.
